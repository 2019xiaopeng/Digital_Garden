# QCBs Digital Garden v1.3.0 Release Note

发布日期：2026-02-27  
版本类型：正式版（Release）

## 本次重点

- 专注系统交互完成一轮收口：任务页统一“专注预设”路径，减少重复配置与跳转心智。
- 周复盘与专注统计彻底解耦：`WeeklyReview` 专注周度诊断，`FocusInsights` 承载全量成就分析。
- Focus 数据链路稳定：模板、单次 run、统计聚合、双模桥接（Tauri/LAN）已闭环。

## 主要功能更新

### 1) 任务页：专注预设更直觉
- 统一预设来源：最近使用 Top3 + 快捷预设（25m/50m/15m）+ 我的模板。
- 保留“保存当前专注配置为模板”轻量入口，不阻断“保存任务”主流程。
- 新增模板删除（归档）入口；删除后自动清理最近预设脏引用。
- 标签支持逐项删除与“清空标签”快捷操作。

### 2) 周复盘：回归单一职责
- 保留：周总专注、周完成率、学科分布、AI 周诊断。
- 移除：标签/模板占比与 TOP3（避免周页变成全局成就看板）。

### 3) 专注成就页：独立看板上线
- 新增 `FocusInsights` 页面与导航入口（桌面侧栏 + 移动底部导航）。
- 默认 all-time（全量历史），支持 7/30/90 天切换。
- 展示：总累计专注、完成轮次、完成率、标签占比、模板占比、TOP3 标签/模板。
- 时长展示统一为 `h m`，可读性更高。

## 配色与体验
- 图表与关键数据卡统一使用 EVA 色盘：`#88B5D3` / `#2a3b52` / `#FF9900` 及明度梯度。
- 深色/浅色模式保持可读性一致，不引入额外高饱和杂色。

## 存储与数据口径
- Focus 统计事实源：SQLite `focus_runs` / `focus_templates`。
- localStorage 仅用于 UI 态缓存：
	- `eva.focus.filters.v1`（成就页筛选态）
	- `eva.focus.recentPresets.v1`（任务页最近预设 Top3）

## 版本与构建
- 版本号升级到 `v1.3.0`：
	- `package.json`
	- `src-tauri/Cargo.toml`
	- `src-tauri/tauri.conf.json`
- 构建验证：前端与 Tauri 构建通过（若有体积告警，均为非阻断项）。
